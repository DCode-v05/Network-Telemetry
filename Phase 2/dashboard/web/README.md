# Phase 2 Dashboard — "Signal Lab" (React + Vite + ECharts)

An interactive, single-file dashboard for the Phase 2 single-detector benchmark.
Replaces the old Plotly HTML report with a custom-designed React app.

- **Stack:** React 18, [Vite](https://vitejs.dev/), [Apache ECharts](https://echarts.apache.org/) (`echarts-for-react`)
- **Output:** `npm run build` emits **one self-contained `dist/index.html`** (JS + CSS inlined via `vite-plugin-singlefile`) — open it directly, no server needed.
- **Data:** baked in at build time from `src/data.json`, produced by `../export_data.py`.

## Workflow

```bash
# 1. (Re)generate the data snapshot from the latest results CSVs.
#    Use the project's Python env that has pandas (Phase 2/env).
python ../export_data.py            # writes src/data.json

# 2. Install deps (first time only).
npm install

# 3a. Develop with hot reload.
npm run dev

# 3b. Build the self-contained dashboard.
npm run build                       # -> dist/index.html
```

Then open `dist/index.html` in any browser.

## What it shows

- Animated KPI strip (total runs, peak detection rate, cleanest detector, best F1 cell)
- Detector roster + per-anomaly winners
- **Performance Matrix** heatmap with metric + window selectors
- TPR vs FPR grouped bars with a 5% FPR reference line
- Metric-vs-window sensitivity lines
- Detection-latency bars and a normalised capability radar
